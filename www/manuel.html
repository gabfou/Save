Creation de questionaire

La première chose à faire et de créer un nouveaux projet (Projet->Nouveau)

une fois créer le projet est accesible depuis Projet->Ouvrir et suprimable depuis Projet->Suprimer

Remplissage et configuration du questionaire

 Une fois le projet créer le menu configuration devrais s'ouvrir automatiquement sinon il est accecible depuis Project->Configuration sur Windows et muranoproject->preference sur Mac

Depuis l'onglet general vous pouver décocher l'option Afficher l'index afin que l'utilisateur arrive directement sur le questionaire (pas de page jour 1, jour 2 etc..)

Creation de la pool de personne

Depuis l'onglet Personne vous pouvez créer des groupes (click droit puis nouv eaux) et des personnes (click droit puis nouvelle personne) que vous pouvez ensuite configurer a l'aide de la fenêtre a droite après avoir clické dessus (n'oubliez pas d'appuyer sur enregistrer)

Les groupe possédent 2 parametre

  1. Noms: c'est le noms qui apparaitera dans la sortie
  2. Description: c'est le noms qui apparaitera sur le formulaire

Les personne posséde 3 parametre

  1. Noms
  2. Prenoms
  3. Email

Creation de la pool de question

Depuis l'onglet question vous pouvez créer des groupes (click droit puis nouveaux) et des questions (click droit puis nouvelle question) que vous pouvez ensuite configurer a l'aide de la fenêtre a droite après avoir clické dessus (n'oubliez pas d'appuyer sur enregistrer)

Les groupe possédent 3 parametre

  1. Noms: c'est le noms qui apparaitera dans la sortie.
  2. Description: c'est le noms qui apparaitera sur le formulaire.
  3. type: définit la façons dont sera affiché le groupe coté formulaire.

 

Les question possédent plusieurs paramétre

  1. Type: Cest le type de question :
    1. nombre: Une question dont la réponse est un nombre.
    2. Oui/non: Une question dont la réponse est oui ou non .
    3. option: une question dont la réponse peut etre plusi eurs option sélectioner a l'avance.
    4. option avec valeur: une question dont la réponse peut etre plusieurs option sélectioner a l'avance corespondant chaqu'une à un chiffre.
    5. suggestion: une question dont la reponse et un text libre.
  2. Noms: c'est le noms qui apparaitera dans l a sortie.
  3. Description: c'est le noms qui apparaitera sur le formulaire.
  4. Min Max: les chiffres maximum et minimum autorisé po ur les question de type nombre.
  5. Option: les options pour les questions de type option ou option avec valeur.
  6. Ponderation: Permet de preparer un coeficient pouvant être appliquer à cette question si elle est numerique
  7. Groupe Cible: définit qui voit la question
  8. Unité: définit l'unité dans laquelle une question nombre doit etre remplie
  9. Estimation et reel: définit si la question doit apparaitre seulement a l'estimation seulement lors des jour ou tout le temps.

Preview

Depuis l'onglet preview vous pouvez acceder a l'arborecence du questionaire de facons plus visuelle

La mise a jour de cette fenêtre nécessite de fermer et rouvrir le menu configuration

D'ici vous pouver cliquer sur chaque groupe et question directement pour les modifier ainsi qu'échanger deux groupe ou deux question en glissant-déposant (attention quand vous échanger deux groupes les questions qui lui sont attribué ne sont pas échanger avec)

Mail et suivi

Mail automatique

Depuis Serveur et mail->Programmer une serie de sondage vous pouvez progra mer des mail automatique

Les dates valides sont tout les quarts d'heures (XXh00, XXh15, XXh30 ou XXh45)

Vous pouvez sélectioner un groupe de personne visé particulier grâce à l'arbre de gauche

Le mail envoyer corespond a celui écrit à droite un lien y est preécrit a chaque fois que cette fenetre et ouverte (vous pouvez chang er)

L'option ref permet d'envoyer l'estimation

L'option iteration permet de préparer plusieurs sondage d'un seul coup les sondages seront rajouté à la même heure que le premier suivant les jours selectionés, si l'option ref est cocher seul la première iteration sera impacté

Mail a la mano

Le lien est http://etudemurano.alwaysdata.net/directlogin.php?p="id de la personne"&s="noms du sondage"

Les données nécessaire à l'écriture peuvent extrait en excell en clickant sur Serveur et mail->Excell info depuis la page principale

Vous pouvez gerez qui doit remplir quoi depuis Serveur et mail->Gestion des jours (le nombre de jours à remplir comprend les jours déjà remplie)

Resultat

Pour extraire les resultats bruts directement vous pouvez clicker sur Outils->Extraire les données brutes en xlsx

Sinon a l'aide du menu Affichage vous avez accès a trois tableaux different (le tableaux global a pas encore été corectement retester)

Vous pouvez convertir n'importe lequel de ces tableaux en xlsx en cliquant sur Outils->Convertir en xlsx

L'option Affichage->Ponderation permet d'apliquer le coeficient de pondération

Vous pouver utiliser le paneaux a gauche pour filtrer les information selon les groupe de personne et de question vous pouvez aussi séléctioner l'iteration minimum et maximum si vous voulez regarder les resultats sur une periode particulière
